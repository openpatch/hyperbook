The SQL-IDE element accepts the following arguments:

- **db**: An url to a sqlite file.
- **height**: Height of the editor. Defaults to 600px.

(See: https://github.com/martin-pabst/SQL-IDE).

:::sqlide{height=500}

```mysql Statements.sql

SELECT * from fluss;

```

:::
