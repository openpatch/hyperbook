# Hinweise

Hinweise sind eine gute Art, um die Aufmerksamkeit des Lesers zu gewinnen.

```md
:::alert
Standard
:::

:::alert{error}
Fehler
:::

:::alert{success}
Erfolg
:::

:::alert{info}
Info
:::

:::alert{warn}
Warnung
:::
```
