# Insert Hyperbook Elements

Hyperbook provides many interactive elements through snippets. These make your book more engaging and interactive.

## How to use snippets:
1. Open any markdown file in your book
2. Type `:` to trigger IntelliSense
3. Select a snippet from the list
4. Fill in the parameters

## Popular snippets:
- **:alert** - Add info, success, error, or warning boxes
- **:video** - Embed videos
- **:youtube** - Embed YouTube videos
- **:collapsible** - Add expandable sections
- **:tabs** - Create tabbed content
- **:onlineide** - Add interactive code editors

Try adding an alert to your index.md file:
```
:::alert{info}
Welcome to my Hyperbook!
:::
```
